# Lovable Clone

A Spring Boot application that replicates core features of Lovable, built with modern Java technologies and integrations.

## Overview

This project is a full-stack web application built with Spring Boot 4.0.3, featuring user authentication, data persistence, AI capabilities, and payment processing.

## Tech Stack

- **Backend Framework**: Spring Boot 4.0.3
- **Language**: Java 25
- **Database**: PostgreSQL
- **Security**: Spring Security with JWT Authentication
- **Data Mapping**: MapStruct 1.6.3
- **AI Integration**: Spring AI with OpenAI
- **Payment Processing**: Stripe API
- **Build Tool**: Maven
- **Additional**: Lombok for boilerplate reduction

## Features

- User authentication and authorization using JWT tokens
- Data persistence with Spring Data JPA
- API validation with Spring Validation
- Entity mapping using MapStruct
- OpenAI integration for AI-powered features
- Stripe payment processing
- RESTful API endpoints
- PostgreSQL database support

## Prerequisites

- Java 25 or higher
- Maven 3.6+
- PostgreSQL database
- API Keys:
  - OpenAI API key
  - Stripe API key
